在div注册事件, 注意 **tabindex** 是重点
```js
<template>
  <div @keyup="renderKeyDown($event)" tabindex="0"></div>
</template>

<script>
  methods: {
    renderKeyDown (e) {
      console.log('e.keyCode' + e.keyCode)
    }
  }
</script>
```
